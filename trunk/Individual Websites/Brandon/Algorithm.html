Algorithm

Huffman Conding

Huffman Coding is a type of Entropy Coding algorithm.
It compresses a file by changing all values in the file into shorter representations of each of those values, in such a way that is is reversable.
In order to find out the values to change each value in the data file, it finds all unique values in the file and allocates a new value to that unique value. It allocates shorter new values for vlaues that occur most often in the file. It the generates a table to show what each unique value in the file is replaces to. The program also has to save this table to the file in order for the process to be reversable.
